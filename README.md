# 🐄 Mon Cheptel

**Mon Cheptel** est une application web moderne de gestion de troupeau. Elle permet le suivi, la gestion et l'administration de données d'élevage de manière simple et intuitive.

Cette application est construite avec des technologies modernes pour assurer performance, sécurité et maintenabilité.

## 🚀 Fonctionnalités principales

- **Gestion des animaux** : Suivi complet de chaque individu du cheptel.
- **Tableaux de bord** : Vue d'ensemble des statistiques de l'élevage.
- **Interface intuitive** : Une interface utilisateur claire, rapide et responsive conçue pour un usage quotidien.

## 🛠️ Stack Technique

- **Frontend / Backend** : [Nuxt 4](https://nuxt.com/) (Vue.js)
- **Base de données** : [PostgreSQL](https://www.postgresql.org/)
- **ORM** : [Prisma](https://www.prisma.io/)
- **Styling** : [Tailwind CSS](https://tailwindcss.com/)
- **Déploiement** : Docker & Docker Compose (via GitHub Container Registry)

## 📦 Structure du projet

Le projet suit la structure standard d'une application Nuxt 4 :

- `/pages` : Vues et routes de l'application.
- `/components` : Composants Vue.js réutilisables.
- `/server` : Routes d'API backend (Nitro).
- `/prisma` : Schéma de base de données et migrations.
